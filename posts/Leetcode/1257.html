1275. Find Winner on a Tic Tac Toe Game

March 23, 2024

描述：

ooxx遊戲，規則就是你想的那樣

給一二維陣列int[][]，回傳字串包含"A", "B", "Pending", "Draw"

若沒有贏家則回傳"Draw"

若遊戲沒有結束則回傳"Pending"

範例：

Example 1: 
x| | 
 |x| 
o|o|x
Input: moves = [[0,0],[2,0],[1,1],[2,1],[2,2]]
Output: "A"
Explanation: A wins, they always play first.
Example 2: 
x|x|o
x|o| 
o| | 
Input: moves = [[0,0],[1,1],[0,1],[0,2],[1,0],[2,0]]
Output: "B"
Explanation: B wins.

解題想法：贏家最快產生是第5局，所以小於5的數字都是未完成。大於5才有需要做判斷是否有贏家;利用計分機制判斷是否有贏家

  • 將列，行，對角線，反對角線個別計分
  • 列與行可以以長度為3的Array作為計分板
  • 對腳與反對腳int作為計分
  • 對角判斷行與列位置相同就是對角 
    o| |
 |o|
 | |o
  • 反對角判斷行與列位置相加為2就是反對角 
     | |o
 |o| 
o| |

流程

  1. 判斷棋局是否已滿5局，若未滿回傳"Pending"
  2. init 行，列，對角線，反對角線計分
  3. 迭代棋局，將玩家分為1, -1，正負只是要區分玩家，最終判斷是否計分達到3，即為某一方勝利
  4. 每次迭代計算該局的行，列，對角出現次數
  5. 判斷該局的行，列，對角是否為3，若達到3，回傳該玩家

public class Solution {
    public string Tictactoe(int[][] moves) {
      if(moves.Length < 5)return "Pending";
        int[] rows = new int[3];//[1, 2, 3]
        int[] cols = new int[3];//[1, 2, 3]
        int diag = 0;
        int antiDiag = 0;
        for (int i = 0; i < moves.Length; i++) {
            int player = i % 2 == 0 ? 1 : -1;//B : A
            int[] move = moves[i];//[0 ,1]
            int row = move[0];// 0
            int col = move[1];// 1

            rows[row] += player;//[1, 0, 0]
            cols[col] += player;//[0 ,1 ,0]
            if (row == col) {
                diag += player;
            }
            if (row + col == 2) {
                antiDiag += player;
            }

            if (Math.Abs(rows[row]) == 3 || Math.Abs(cols[col]) == 3 || Math.Abs(diag) == 3 || Math.Abs(antiDiag) == 3) {
                return player == 1 ? "A" : "B";
            }
        }

        return moves.Length == 9 ? "Draw" : "Pending";

    }
}
