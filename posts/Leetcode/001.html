001. Two Sum

April 08, 2024

描述：

給一個整數Array nums 和一個整數目標值target target，傳回兩個數的index ，使得它們相加總和等於target。

可以假設每個輸入只對應一種答案，不能重複利用陣列中同樣的元素。

可以以任意順序 return答案。

範例：

Example 1: 
Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].
Example 2: 
Input: nums = [3,2,4], target = 6
Output: [1,2]

解題想法：暴力解法，遍歷整個Array，計算每個數字與目標值的差值，找出與目標值相等的數字，回傳對應的index

流程

  1. 迭代nums ，相加當下位置 i與當下位置以後j 每個數字直到相加總和等於target
  2. 兩數相加，與目標值比較，若相等，回傳對應的 index 

public class Solution {
   public int[] TwoSum(int[] nums, int target) {
    for(int i = 0; i < nums.Length; i++) {
        for(int j = i + 1; j < nums.Length; j++) {
            if(nums[i] + nums[j] == target) {
                return new int[] { i, j };
            }
        }
    }
    return new int[] { };
  }
}

解題想法：使用HashMap記錄已經遍歷過的數字，以數字為key，index為value，利用與目標值相減的結果，找出與目標值相等的數字回傳index

流程

  1. init Dictionary作為記錄已經遍歷過的數字位置
  2. 把遍歷過的數字加入Dictionary，數字為key，位置為value
  3. 每次遍歷與target 相減，利用相減的結果比對出現過的值(ContainsKey())，如果有相符的數字，取其value加入Array回傳。 

public class Solution {
    public int[] TwoSum(int[] nums, int target) {
      Dictionary map = new Dictionary();

      for(int i = 0; i < nums.Length; i++)
      {
        int j = target - nums[i];
        if(map.ContainsKey(j))
        {
          return new int[]{map[j], i};
        }
        map[nums[i]] = i;
      }

      return new int[]{};
    }
}
